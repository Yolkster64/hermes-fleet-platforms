# Hermes Fleet: Best GUI & Animation Frameworks (2026)

## C#
- **WPF:** Rich animation, enterprise, hardware-accelerated, modern UX (Visual Studio).
- **MAUI:** Cross-platform, modern UI, good animation, hot reload.
- **Avalonia:** Open-source, cross-platform, XAML, good animation.
- **Uno Platform:** C#, XAML, runs on Windows, WebAssembly, Linux, macOS, iOS, Android.

## JavaScript
- **Electron:** Web tech, flexible, CSS animation, heavy (VS Code, Slack).
- **Tauri:** Lightweight, system webview, Rust backend, fast startup.
- **NodeGUI:** Native widgets (Qt), JS/React API, better perf than Electron.

## C++
- **Qt:** Powerful, animation-rich, cross-platform, modern UI (Maya, VirtualBox).
- **wxWidgets:** Native controls, basic animation, lightweight (Audacity).
- **Dear ImGui:** Dev tools, game editors, real-time UIs, manual animation.

## Hybrid
- **Flutter:** Dart, cross-platform, rich animation, modern UX.

## Notable Combos
- C# + WPF: Windows-only, rich animation, enterprise.
- C# + Avalonia: Cross-platform, modern look.
- C++ + Qt: High-perf, animation-rich, cross-platform.
- JS + Electron: Web devs, rapid prototyping, cross-platform.
- Flutter: Modern, cross-platform, rich animation.
