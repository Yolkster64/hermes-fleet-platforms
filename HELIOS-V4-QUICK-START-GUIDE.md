## GLOSSARY

- **Availability:** Percentage of time system is operational (target: 99.85%)
- **ROI:** Return on investment (revenue generated per dollar spent)
- **SLO:** Service Level Objective (performance target)
- **ML:** Machine Learning (artificial intelligence)
- **DDoS:** Distributed Denial of Service (attack type)
- **SOC 2:** Security audit standard (System and Organization Controls)
- **ISO 27001:** Information security management standard
- **Compliance:** Adherence to regulations and standards
- **Penetration Testing:** Authorized security attack simulation
- **Auto-Scaling:** Automatic resource adjustment based on demand
