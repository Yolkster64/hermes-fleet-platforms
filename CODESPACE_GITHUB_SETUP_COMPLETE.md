
### Step 3: Configure GitHub Secrets (3 minutes)
```
Go to: Settings → Secrets and variables → Actions

Add 4 required secrets:
- AZURE_SUBSCRIPTION_ID
- AZURE_TENANT_ID
- AZURE_CLIENT_ID
- AZURE_CLIENT_SECRET
```

```

gh workflow run deploy.yml -r main -f tier=enterprise

Monitor:
gh run watch
```
