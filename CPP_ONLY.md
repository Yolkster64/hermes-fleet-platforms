# Hermes Fleet: C++-Only Variant

## Design
- C++ for all logic and GUI (Qt/wxWidgets/Native)

## Pros
- Max performance, resource efficiency, direct OS/hardware access

## Cons
- Slower dev, complex UI, harder distribution

## Best for
- Real-time, embedded, perf-critical

## Integration
- Expose APIs via sockets, REST, gRPC for interop

## Comparison
- Best perf, hardest dev/maintenance
