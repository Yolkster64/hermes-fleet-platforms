## 🚀 QUICK START

### Developers (5 Minutes)
```bash
git clone https://github.com/M0nado/helios-platform
cd helios-platform
dotnet restore && dotnet build
dotnet run
# Open http://localhost:5000/api/v1/health
```

### DevOps (5 Minutes)
```bash
kubectl apply -f k8s/secrets.yaml
kubectl apply -f k8s/deployment.yaml
kubectl get pods
kubectl logs deployment/helios-backend -f
```

### Operators (2 Minutes)
```bash
cd src/phases
.\master-deploy.ps1
# Platform ready!
```
