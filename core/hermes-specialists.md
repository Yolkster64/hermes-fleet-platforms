# Hermes Specialist Agents

This document catalogs specialized Hermes agents, each with unique skills and hybrid roles. Example agents:
- DataSavant: SQL/data mining, analytics
- Visionary: Image/3D/GUI generation
- Polyglot: Multilingual LLM, translation
- Sentinel: Security, anomaly detection
- Synthesist: Hybrid C++/Python/C# orchestration

Agents can teach, challenge, and learn from each other and external data.
