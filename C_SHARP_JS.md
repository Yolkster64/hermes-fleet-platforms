# Hermes Fleet: C# + JavaScript Variant

## Design
- C# backend/API/desktop, JavaScript frontend (Electron/Blazor/web)

## Pros
- Modern UI, cross-platform, large dev pool, rapid dev

## Cons
- Two stacks, heavier apps, more deployment complexity

## Best for
- Web-like UIs, dashboards, API-driven systems

## Integration
- REST/gRPC/SignalR for backend-frontend comms
- Electron/Blazor for desktop/web

## Comparison
- Easier UI than C++-only, more complex than C#-only
