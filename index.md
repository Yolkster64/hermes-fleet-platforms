## 💻 Deployment Options

### Option A: GitHub Codespace (Recommended)
1. [Launch Codespace](https://github.com/codespaces/new?repo=M0nado/helios-platform)
2. Wait 3-5 minutes
3. Run: `gh workflow run deploy.yml -r main -f tier=enterprise`
4. Monitor: `gh run watch`

### Option B: Local Deployment
```bash
git clone --recurse-submodules https://github.com/M0nado/helios-platform.git
cd helios-platform
dotnet add package HELIOS.Platform
gh workflow run deploy.yml -r main -f tier=enterprise
```

### Option C: NuGet Package
```bash
dotnet add package HELIOS.Platform
# Then use in your code
```
