## 🛠️ TROUBLESHOOTING

### Issue: GitHub CLI Not Authenticated
**Solution:** Run `gh auth login` and follow prompts

### Issue: Codespace Won't Load
**Solution:** 
- Refresh the page
- Check GitHub status: https://githubstatus.com
- Try different browser

### Issue: Workflow Secrets Not Found
**Solution:**
- Verify secrets added to Settings → Secrets
- Check secret names match exactly
- Secrets are case-sensitive

### Issue: Deployment Fails
**Solution:**
- Check logs: `gh run view [run-id] --log`
- Verify all secrets configured
- Check GitHub Actions usage limits

### Issue: Pages Site Not Building
**Solution:**
- Wait 5 minutes after enabling Pages
- Check Actions tab for build job
- Verify _config.yml syntax

**For More Help:** Check COMPLETE_GITHUB_SETUP_GUIDE.md
