## 🚀 How to Enable (2 Minutes)

### Quick Steps

1. **Go to Settings**
   - URL: https://github.com/M0nado/helios-platform/settings/pages

2. **Select Source**
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/" (root)

3. **Save**
   - Click the Save button

4. **Wait**
   - GitHub will build and deploy (30-60 seconds)
   - You'll see a green checkmark when complete

5. **Visit Site**
   - https://m0nado.github.io/helios-platform

### What Happens Automatically
- GitHub processes all .md files
- Applies Slate theme styling
- Generates HTML versions
- Publishes to CDN worldwide
- Site goes live automatically
