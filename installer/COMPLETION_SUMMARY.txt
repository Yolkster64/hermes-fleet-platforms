QUICK START
================================================================================

Load the framework:
  . C:\HELIOS\migration-orchestrator.ps1

Create a migration:
  $m = New-MigrationOrchestrator -Name "MyMigration" -FromVersion "v1" -ToVersion "v2"

Configure transformation:
  $m.AddFieldMapping("old_id", "id", "int")
  $m.AddFieldMapping("old_name", "name", "string")

Add validation:
  $m.AddValidationRule("RequireId", {
    param($data)
    return @{ Passed = $data.id -gt 0; Message = "ID required" }
  })

Register upgrade:
  $m.RegisterUpgrade("v1", "v2", {
    param($data)
    $data['Version'] = 'v2'
    return $data
  })

Execute migration:
  $result = $m.ExecuteMigration(@{ old_id = 123; old_name = "Test" })

Check results:
  Write-Host "Success: $($result.Success)"
  Write-Host "Backup: $($result.BackupPath)"

Cleanup:
  $m.Dispose()
