# Hermes Fleet: C#-Only (MAUI/WPF) Variant

## Design
- C# for all logic and GUI (MAUI for cross-platform, WPF for Windows)

## Pros
- Unified stack, rapid dev, strong .NET/Azure integration, cross-platform (MAUI)

## Cons
- Lower performance, less low-level access, .NET runtime required

## Best for
- Enterprise, business, rapid prototyping

## Integration
- Can interop with C++ if needed, but not required
- Self-contained deployment possible

## Comparison
- Faster dev than C++-only, less flexible than mixed/hybrid
