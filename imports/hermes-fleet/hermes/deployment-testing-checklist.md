# Hermes Deployment Testing Checklist

- [ ] Test clean install on all supported platforms (local, cloud, hybrid)
- [ ] Test upgrade from previous versions
- [ ] Test rollback and recovery
- [ ] Validate device-specific drivers/utilities (Intel Arc, NVIDIA, THX, Steam, Synapse/Chroma)
- [ ] Simulate failure/recovery scenarios
- [ ] Log and review all deployment actions
- [ ] Confirm profile migration and settings
- [ ] Security validation (signatures, RBAC, Azure Vault integration)
- [ ] User acceptance testing (all profiles)
