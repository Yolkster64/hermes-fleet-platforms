## 🚀 INSTALLATION & USAGE

### Quick Start
```bash
git clone https://github.com/M0nado/helios-platform.git
cd helios-platform
dotnet build -c Release
dotnet test -c Release
dotnet publish -c Release -r win-x64
```

### System Requirements
- Windows Server 2019+ or Windows 11+
- .NET 8 SDK/Runtime
- 4+ GB RAM (8+ recommended)
- 5+ GB disk space

### Three Deployment Options
1. **MSI Installer** (Recommended)
2. **Manual Installation** (Advanced)
3. **Docker** (Cloud-native)

See `INSTALLATION_GUIDE.md` for detailed procedures.
