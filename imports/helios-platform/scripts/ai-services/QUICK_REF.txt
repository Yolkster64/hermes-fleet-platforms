## 🚀 Quick Commands

```powershell
# Initialize hub
. .\scripts\ai-services\hub.ps1

# Request analysis
$result = $hub.RequestAnalysis("code-review", $code)

# View results
$result.CombinedResult

# Check stats
$hub.GetUsageStats()
$hub.GetHealthStatus()

# View usage report
.\scripts\ai-services\view-ai-usage.ps1

# Check costs
.\scripts\ai-services\show-ai-costs.ps1 -ShowTrends

# Run tests
.\scripts\ai-services\test-ai-services.ps1

# Configure
.\scripts\ai-services\configure-ai-services.ps1
```

## 🆘 Troubleshooting

| Issue | Solution |
|-------|----------|
| API key failed | Run `validate-api-keys.ps1 -Interactive` |
| Rate limit | Check `view-ai-usage.ps1`, adjust in `configure-ai-services.ps1` |
| High costs | Run `show-ai-costs.ps1 -ShowTrends`, review service usage |
| Connection failed | Run `test-ai-services.ps1 -Verbose` |
| No log output | Check log path in `ai-services-config.json` |
