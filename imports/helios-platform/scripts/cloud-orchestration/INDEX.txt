🚀 QUICK START GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Initial Setup:
   PS> .\hybrid-identity\setup-hybrid-identity.ps1 -EnvironmentType FullSync
   PS> .\hybrid-identity\manage-hybrid-auth.ps1 -AuthMethod All

2. Start Synchronization:
   PS> .\orchestrator\sync-local-to-cloud.ps1 -Environment Production -SyncType Full

3. Monitor Environment:
   PS> .\orchestrator\monitor-both-environments.ps1 -CheckHealth All -ContinuousMonitoring

4. Get Recommendations:
   PS> .\ai-cloud\ai-recommendations.ps1

5. Generate Compliance Report:
   PS> .\compliance\compliance-reports.ps1 -ReportPeriod Weekly

Common Issues & Solutions:

  Sync Failing:
    • Check network connectivity
    • Verify Azure credentials
    • Review logs in ./logs/ folder
    • Check configuration file settings

  Failover Not Triggering:
    • Verify health thresholds
    • Check failover policy settings
    • Review environment health status
    • Check alert configuration

  Authentication Issues:
    • Run setup-hybrid-identity.ps1
    • Verify manage-hybrid-auth.ps1
    • Check AD connectivity
    • Review Azure AD settings
